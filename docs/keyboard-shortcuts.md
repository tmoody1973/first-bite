# First Bite — Keyboard Shortcuts

## Journey Viewer

| Key | Action |
|-----|--------|
| `→` (Right Arrow) | Next stop |
| `←` (Left Arrow) | Previous stop |
| `Space` | Next stop |
| `P` | Jump to poster/video (last slide) |
| `End` | Jump to poster/video (last slide) |
| `Home` | Jump to first stop |

## Navigation

- **Tap right third of screen** → Next stop
- **Tap left third of screen** → Previous stop
- **Auto-advance** → Moves to next stop when narration finishes

## Demo Recording Tips

1. Pre-generate a journey 10 min before recording
2. Use `→` to quickly tap through stops
3. Press `P` to jump straight to the poster/video finale
4. Press `Home` to go back to stop 1
